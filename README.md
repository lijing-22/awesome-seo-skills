# Awesome SEO Skills

Agent skills for practical SEO workflows powered by AIsa.

This repository collects reusable skills for AI coding agents, SEO operators, and growth teams. Each skill is designed to turn live SEO data, search-result intelligence, and large language models into concrete SEO decisions.

## Skills

| Skill | Purpose |
|---|---|
| `seo-keyword-research` | Crawl a site first, infer its product and business direction, then build an SEO keyword strategy using AIsa DataForSEO endpoints and AIsa LLM analysis. |

## Requirements

Most skills require an AIsa API key:

```bash
export AISA_API_KEY="your-aisa-api-key"
```

Use the AIsa API for both live data and LLM reasoning:

- Data APIs: `https://api.aisa.one/apis/v1/...`
- LLM gateway: `https://api.aisa.one/v1/chat/completions`
